# 3-安裝開發環境

預計學習時間: 2小時

# 3-1 確認操作系統

請確您的 Mac 操作系統為 10.10 ( Yosemite ) 以上

點擊左上角蘋果圖示，選擇『關於這台 Mac』，就可以看到自己電腦的設定。

# 3-2 安裝 Command Line Tools

先點選右上角的搜尋

搜尋 Terminal

打開你的 “Terminal” （終端機）程式。（未來我們都會透過這個程式來輸入程式指令）

在終端機裡輸入指令：

xcode-select --install

他將會安裝 command line tools ( 這是安裝 Ruby 所需要的 Library）

執行後會出現這個視窗，點選“安裝”。

# 如何確定已經裝好？

在終端機輸入：

xcode-select -p

顯示出： /Applications/Xcode.app/Contents/Developer

即是成功安裝！

# 什麼是Command Line Tools？

一種可以編譯程式的軟體。
但我們不會直接使用它，你可以把它想成是電腦執行程式需要的軟體工具跟套件之一就可以了。

# 程式碼要打在哪裡？

Terminal（終端機）
這裡是輸入所有指令的地方。
當執行中請不要隨便關閉，避免執行錯誤。

# 3-3 安裝 Homebrew

# 如何安裝 Homebrew？

在終端機輸入：

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

此命令較長，請複製貼上在終端機然後 Enter。 若執行失敗，請以官網最新的安裝方式為主。

執行時會出現很多訊息，按照要求輸入鍵盤上的 Enter 鍵同意即可。

期間會被問到系統密碼，請直接輸入密碼，然後按 Enter 即可。

（你會發現輸入密碼時畫面沒有任何反應，但其實電腦已經有收到訊息了）

如果沒有輸入 Enter 鍵，而是按了別的按鍵，則安裝過程可能會被中斷。需再重新輸入安裝命令，繼續安裝，直到屏幕顯示安裝成功。

# Homebrew 是什麼？

Homebrew 是開發者社群專為 MacOS 打造的第三方套件管理程式，比如說安裝 Ruby 所需的一些編譯工具或函式庫，就可以從 Homebrew 取得。

# 提交作業

安裝完成後請輸入“ok”

# 3-4 安裝 git

# 安裝 Git

使用 Homebrew 安裝 Git
在終端機裡輸入指令：

brew install git

# 怎麼知道自己安裝成功？

在terminal輸入

git --version

會顯示：git version 2.27.0（數字依照各版本可能會不同）

# Git是什麼？

Git 是目前最流行的原始碼版本控制軟體。
你可以把 Git 想像是程式碼的時光機。

本節作業

Rails 環境建置作業：安裝 git

# 提交作業

請在終端機執行 git --version

在這裡貼上你的 git 版本號

# 3-5 安裝 ImageMagick

在終端機裡輸入指令：

brew install imagemagick

# ImageMagick 簡介

ImageMagick 是專門處理圖片的函式庫。

未來我們在 Ruby 中常常需要使用 ImageMagick 相關的功能，所以通常在初始環境時，我們也會一併先安裝。

# 3-6 安裝 PostgreSQL

在終端機裡輸入指令：

brew install postgresql

安裝完畢後：

brew services start postgresql

（會確保在你下次開機時，也一併啟動資料庫）

# PostgreSQL 簡介

PostgreSQL 是一套強大的資料庫。許多網站開發預設使用 PostgreSQL 作為底層資料庫。在初始環境時，我們也會一併先安裝。

# 3-7 安裝 RVM

請在終端機輸入：

\curl -sSL https://get.rvm.io | bash -s stable

安裝完再輸入：

source ~/.rvm/scripts/rvm

讓 rvm 生效。

# RVM 簡介

在 Mac 開發 Ruby on Rails 程式時，因為 Ruby 版本更新很快，常常會遇到一些版本與權限上的問題。所以當在開發時，我們通常不會使用系統內建的 Ruby。而是會改採用一套 Ruby Version Manager 去管理 Ruby 的安裝與升級。透過 Ruby Version Manager (RVM) 去安裝 Ruby。

RVM 的官方網址在 http://rvm.io

# 3-8 安裝 Ruby

請在終端機輸入：

rvm install 2.7.1

然後輸入：

rvm use 2.7.1 --default

使用 ruby 2.7.1 為預設版本 (可改自己需要的版本)

安裝完成後需要重開 Terminal。(在 Terminal 界面按 command + Q 退出，然後重新打開 Terminal)

rvm list

這個指令可以列出目前已安裝的 Ruby 版本。

# Ruby是什麼？

一種程式語言，它經常和框架（Rails）結合，常用於快速開發。
你將會在其他課程中學習到如何使用 Ruby on Rails 來搭建網站。

# 3-9 安裝 Rails

請在終端機輸入：

gem install rails -v 6.0.3.2

後面的參數是指定版本，預設會抓最新版。

# Rails是什麼？

你將要學習使用的開發框架。

# 安裝失敗怎麼辦？

失敗的可能原因是 gem 或 xcode-select 不是最新版本
可以使用這兩個指令來更新

gem update --system

xcode-select --install

# 提交作業

請在終端機執行 rails -v

並在這裡貼上顯示的版本號
