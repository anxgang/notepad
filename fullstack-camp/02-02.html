# 2-Ruby 程式語言

預計學習時間: 1小時

# 2-1 基礎 Ruby

# 目標

  • 會使用 Ruby 基本的語法
  • 用 IRB 跑 Ruby 程式
  • 做簡單的算術
  • 使用、瞭解 Variable（變數）
  • 使用、瞭解 Array（陣列）
  • 使用 Loop（迴圈）和條件判斷

# 步驟

# 步驟 1

在終端機打這些字來啟動 IRB（Interactive Ruby Shell），這個程式可以讓你試一些 Ruby 程式碼：

irb

長得像這樣（你看到的或許會有些不同）：

2.7.1 :001 >

# 步驟 2

接著試一些 Ruby 內建的簡單算術。在 IRB 裡面打字：

3 + 3
=> 6

7 * 6
=> 42

# 步驟 3

Variables（變數） 指的是一個被指定 values（值）的名字。

my_variable = 5

這會把 5 這個值指定給 my_variable 這個名字。

# 步驟 4

也可以對變數做算術：

my_variable + 2
=>7

my_variable * 3
=>15

# 步驟 5

變數可以放多個值，這叫 Array（陣列）。

fruits = ["kiwi", "strawberry", "plum"]

這裡我們用了 fruits 這個變數來放一群水果名。

# 步驟 6

在終端機打這些字：

fruits = fruits + ["orange"]
fruits = fruits - ["kiwi"]

  • + 和 - 叫做 Operators（運算符號）。跟數字一樣，我們也可以把它們用在 array 上。

# 步驟 7

在 Ruby 裡面，萬物皆有 class （類別）。請在 IRB 裡面打字：

7.class
"kiwi".class
fruits.class
這些就是我們至今介紹的三個資料型態：

Integer（數值）、String（文字）、Array（列表）。

# 步驟 8

每個 class 都有不同的 method(方法) 可以在該 class 的 instance(實例) 使用

fruits.length
fruits.first

你可以看看一個 object 所擁有的所有 method：

fruits.methods

還可以串在一起：

fruits.methods.sort

# 步驟 9

Array 有個 method 叫做 each，會走訪它裡面每一個項目並執行程式。

fruits.each do |fruit|
  puts fruit
end

這會把 fruits array 裡面的 第一項（"strawberry"） 拿出來，把它設給絕對值符號中的 fruit 變數，並執行 do 和 end 之間的程式。然後對其他項重覆做這件事。上面這段程式會印出水果的列表。

# 步驟 10

conditional（條件式） 只會在條件滿足的時候才執行程式。

if my_variable > 1
  puts "YAY!"
end

如果 my_variable 變數里面儲存的值大於 1，才會印出 YAY!。

請試著把 > 改成 <。

# 步驟 11

你也可以做自己的 method：

def pluralize(word)
  word + "s"
end

pluralize("kiwi")

Method 會取 parameters（參數），表示它們要對哪些變數操作。以此例而言，我們做了一個 method 叫做 pluralize ，它取一個參數 word，是一個單字。

Method 也可以 return（回傳）資料。以此例而言，pluralize 會回傳 word 單字加上 's' 結尾。在 Ruby 裡面，method 會回傳最後一行的結果，無論該結果是什麼。

# 步驟 12

離開 IRB
輸入 exit 然後 Enter 即可！
