# 5-選購 Mac 電腦

# Mac電腦的最低需求

為了確保開發過程的流暢。我們建議購買如下機器配置：

  • 2014 年以後出廠的 MacBook Air / MacBook Pro / iMac / Mac Mini
  • 屏幕限 13" 以上，最好是 15"+
  • 內存限 8GB 或以上，最好是 16GB
  • 操作系統限 OSX 10.10 以上

# 為什麼一定要用Mac

  • 我們教材均以 Mac 系統為基礎編寫
  • 若你真心要將程式設計當作一門認真的事，那麼一定要使用 Mac 寫程式
  • Mac上有著開源世界所有最好程式員提供的各式工具以及最佳配置，可以讓你至少躲掉一百個以上的坑

# 購買通路

  • 蘋果官網
  • 蘋果官網(教育價) - 如果是學生/老師可以用教育價購買
  • 蘋果官網(整修品) - 官方認證整修品，也許可以撿到便宜，但要注意看一下出廠年份
  • Pchome - 有時候有折扣，也可搭配信用卡折價購買

提醒小心 2016~2018 年出廠的 macbook，鍵盤用久可能有入塵問題，請自行爬文

# 其他硬體（可選）

這裡有幾樣東西，我們也希望你額外準備

  • 一個外接屏幕，22吋或以上（1080P分辨率或以上）都可以，看程式碼的時候會方便很多
  • 一把好的機械式鍵盤，幫助提高輸入效率

開發之前的準備：選購Mac電腦

寫下你購買電腦的型號、內存。

# 本節常見問題

# 需要購買最新款macbook 嗎？

可以，但新款macbook只有12吋，強烈推薦配一台外接顯示器。

# 不能使用黑蘋果嗎？

如果用黑蘋果（即Windows電腦加虛擬機）的話，極可能會遇到各種問題而無法繼續學習。

# 需要安裝雙系統嗎？

如果只是全棧營的課程，基本上不需要安裝雙系統。實在需要進行一些 Windows 系統下的操作，可以安裝 Paralles 或 VMware 虛擬機以後，再安裝一套 Windows 系統。

# 請問全棧營對 mac 的配置的最低要求是什麼？512G 的 Mac 夠用嗎？

8G內存的Mac 電腦，2014年及之後生產的。128G/256G/512G/1TD都是指硬盤容量，一般256G或以上都可以。

# 蘋果電腦一直插著電源用，還是充滿電後拔了用？

建議一直插電源用。

# 使用無線的外接鍵盤嗎？

可以，尤其是新款 Macbook Pro 只有 USB Type-C 接口，藍牙鍵盤比較合適。

# 能不能用iPad？

只能拿來玩 Swift Playgrouds，無法用來學習全棧營課程。
