# 4-設定編輯器 & 終端機 & Shell

預計學習時間: 20分鐘

# 4-1 安裝編輯器-VSCode

在撰寫程式的時候，我們還需要能夠編輯程式的軟體。 目前市面上有許多各種不同的 Rails 編輯器，這裡我們推薦給新手的是 VSCode。

# VSCode

由 Microsoft 維護的 VSCode

下載後，將 vscode 拖曳至 “應用程式” 即可完成安裝。

編按

原全棧營推薦的是 Atom，但近年大家愛用的是 VSCode，為避免以後還要多學一個編輯器，請直接使用 VSCode 吧！

# 設定 Terminal 內可開 VSCode

# 如何在 Terminal 內可以用慣用的編輯器 ”快速打開“ 目前正在做的專案（project）呢？

這個功能叫 shell command。
這個功能可以幫助你在最短的時間內，開啟正在執行的專案程式碼檔案。

# 安裝 VSCode 的延伸套件

安裝 ruby 語言
點選 “延伸套件” 圖示，並搜尋 ruby 並按下 install

之後如有其他需求，可以再試試看安裝其他延伸套件。

# 4-2 安裝終端機 - iTerm2

iTerm2 是一個可以用來取代 MacOS 原生 Terminal 工具，
擁有許多更方便的功能和特色，可至官網下載安裝。

# 調整佈景主題

可按下 cmd+t 到 Profiles > Colors > 右下角的 Color Presets 選擇顏色主題

如果都不喜歡，也可以找別人設定好的主題來匯入 iTerm2-Color-Schemes

# 調整字型

建議要更換為等寬字，可以下載此字型 Meslo

並到偏好設定套用字體 Meslo

Profiles > Text > Change Font

其他字體可至 Powerlinefonts@github 下載
下載方式：點進想下載的字體 xxx.ttf ，並點選『Download』

# iTerm2 快速鍵

  • cmd + t 開啟新分頁
  • cmd + k 清空螢幕
  • ctrl + tab 切換分頁

# 4-3 安裝 Shell - zsh

zsh 是一個可以用來取代 MacOS 原生的 bash 指令。 因為 zsh + oh-my-zsh 的設定集能夠讓畫面更美觀，並擁有許多額外如 auto-complete 等功能。

# 安裝 zsh

打開 Terminal 輸入

brew install zsh zsh-completions

確認安裝成功

zsh --version

接著要將預設 shell 改為 zsh，一樣在終端機輸入

sudo sh -c "echo $(which zsh) >> /etc/shells"
chsh -s $(which zsh)

然後重新開啟你的 Terminal ，輸入

echo $SHELL

如果有成功變更為 zsh 的話應該會看到 /usr/local/bin/zsh。

# 安裝 Oh My zsh

Oh My Zsh 是一個用來管理 zsh 設定檔（configuration）的框架，提供了很多的外掛（plugin）和主題（theme）可以選擇。

輸入指令安裝

sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

接下來我們要套用主題，輸入

code ~/.zshrc

接著會打開 zsh 的設定檔，找到 ZSH_THEME，將這段的內容改成下面這段後存檔

ZSH_THEME="agnoster"

安裝完成後，重新打開 iterm2，外觀會類似這樣

若需要查看其他主題，可以到 oh-my-zsh Themes 查看
