# 4-參加線下/線上 Meetup

# Meetup是什麼？

Meetup 是一種具有相似興趣的人舉辦面對面的活動，旨在幫助大家找到彼此，共同學習、分享、幫助、鼓勵與提升。

# 為什麼要 Meetup？

  1. 學習是一種社交活動（Meetup 可以突破一個人學習時的無助與孤獨感）；
  2. 找到同類和群體，收穫友誼，精神世界更豐富；
  3. 訂立學習計畫、交流和分享，收穫專業技能；
  4. 帶領周圍的人一起共同成長。

# Meetup 怎麼做？

  1. 每週舉行一次；
  2. 每次 3 ~ 5 小時；
  3. 開始時，每人分享 3 ~ 5 分鐘（關於本週學到的最棒工具/概念以及最大的坑）；
  4. 剩下的時間做作業，不懂的時候及時詢問身邊的同學（教是最好的學）；
  5. meetup 結束後及時寫 blog（對相關知識進行複習、練習）。
