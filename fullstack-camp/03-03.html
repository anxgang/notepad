# 3-建立一個討論群組

預計學習時間: 1小時以內

# 3-1 建立討論群組的架構

# 目標

在這一章，我們要帶大家設計出一個討論群“列表”。我們會完成以下事項：

  • 討論群要有“標題”與“敘述”
  • 使用者要可以看到“討論群一覽表”

# 步驟

# Step 0. git checkout -b ch02

因為我們現在要實作第二章的內容，新增一個 branch 把實作內容集中在 ch02 是比較合理的。將來回顧就可以比較清楚 在 ch02我們做了哪一些練習。

git checkout -b ch02

# Step 1. 設計 Group 的 model 架構

為了做到在討論群要有“標題”與“敘述”，我們在這裡要建立一個 model Group，並建立數據表 group 的兩個欄位:

  • title (string 字串屬性)
  • description (text 文字屬性)
rails g model group title:string description:text

將數據庫建立起來，並執行 rake db:migrate

rake db:migrate

提示：每次產生 migration 之後都要執行 rake db:migrate 。

# Step 2. 產生 groups controller

  • 接下來我們要建立一個可以展示“討論群一覽表”的功能
  • 這個地方會是 groups_controller 下的 index action
rails g controller groups

# Step 3. 建立 index action 以及 index 的 view

修改 app/controllers/groups_controller.rb

# app/controllers/groups_controller.rb
class GroupsController < ApplicationController
  def index
    @groups = Group.all
  end
end

touch app/views/groups/index.html.erb

<!-- app/views/groups/index.html.erb -->
<div class="row mt-2">
  <div class="col text-right">
    <%=link_to("New group", new_group_path, class: "btn btn-primary") %>
  </div>
</div>
<div class="row mt-2">
  <div class="col">
    <table class="table table-hover">
      <thead>
        <tr>
          <th scope="col">#</th>
          <th scope="col">Title</th>
          <th scope="col">Description</th>
        </tr>
      </thead>
      <tbody>
        <% @groups.each do |group| %>
          <tr>
            <td>#</td>
            <td><%= link_to(group.title, group_path(group)) %></td>
            <td><%= group.description %></td>
            <td>
              <%= link_to("Edit", edit_group_path(group), class: "btn btn-sm btn-secondary") %>
              <%= link_to("Delete", group_path(group), 
                    class: "btn btn-sm btn-secondary", 
                    method: :delete, data: { confirm: "Are you sure?" }) %>
            </td>
          </tr>
        <% end %>
      </tbody>
    </table>
  </div>
</div>

# Step 4. 在 routing 上掛上 groups

在 root 'welcome#index' 前加上一行 resources :groups

# config/routes.rb
Rails.application.routes.draw do
  resources :groups
  root 'welcome#index'
end

打開 http://localhost:3000/groups 可以看到以下成果

當然現在沒有東西，空蕩蕩的。等一下讓我們來加一些資料。

# Step 5. 加入一些資料。

首先，打開 rails console

rails console

然後再輸入

Group.create(title: "Board 1", description: "Board 1 body")
Group.create(title: "Board 2", description: "Board 2 body")

再重新打開 http://localhost:3000/groups 可以看到以下成果

# Step 6. git 存檔

輸入 exit 命令 退出rails console

git add .
git commit -m "create groups index"

# 3-2 將首頁換成討論群組一覽表

# 目標

在這一節，我們會完成以下事項：

  • 使用者在首頁，可以直接看到“討論群一覽表”

# 步驟

# Step 1. 修改 routing

把 root 'welcome#index' 刪掉，換成 root 'groups#index'

TIP

記得 + - 不要打進去，他們各代表“加這一行” / “減這一行”的意思。

# config/routes.rb
Rails.application.routes.draw do
  resources :groups
+ root 'groups#index'
- root 'welcome#index'
end

然後你就可以在 http://localhost:3000 看到 Group 一覽表了

# Step 2. git 存檔

git add .
git commit -m "replace root with groups#index"
