# 5-體驗 Rails

預計學習時間: 1小時

# 5-1 體驗：建立一個 Rails 專案

# 目的

建立一個 Rails 專案並且可以在瀏覽器上打開操作。

# 步驟

# 1. 開啟終端機，建立一個新的 Rails 專案

輸入下面指令，確認ruby版本

ruby -v

輸入下面指令，確認rails版本

rails -v

輸入下面指令，建立一個新的 rails 專案

rails new first_app

( 等待安裝完成 ... 中間會跑很多訊息 )

一個新的 Rails 專案就建立完成了！

# 2. 進入專案

在終端機輸入，進入案目錄

cd first_app

提示

後面仍有一些安裝步驟，將開發工具跟第三方服務裝完。才能幫助您用更方便、快速、有效的方式開發。

# 常見指令說明

cd first_app

cd 是切換資料夾用的指令。
假設我們現在有一個專案叫做 first_app ，現在要用 terminal 打開這個資料夾
在 terminal 輸入 cd first_app
你會發現指令前面多了一個 first_app ，這表示你現在的位置在 first_app 這個資料夾裡面。

那麼要怎麼看到資料夾裡面的檔案呢？
輸入 ls -al
這個指令會把資料夾裡面所有的檔案名稱告訴你。
當未來要做任何專案的時候，請務必確認你的位置是在該專案的資料夾下。

如果想要離開這個資料夾，可以執行 cd 來退出。

# 5-2 進入 Rails 專案，並執行 server 環境運作

# 進入 Rails 專案

（如果重啟過電腦，輸入 cd first_app 進入剛建立好的專案）

你可以輸入 ls 來確認目前資料夾內容

# 啟動專案

輸入

rails s

即可啟動 server 模式

看到 Use Ctrl-C to stop 就表示成功了

接下來請用瀏覽器開啟這個網址

http://localhost:3000

即可進到 Rails 的畫面

# 常用指令說明

rails s - 讓你的專案開始執行的指令。 如果想要停止可以按下：ctrl+c。

pwd - 這個指令可以確認你目前所在位置路徑，他會告訴你現在的資料夾在電腦裡的哪個地方。

# 5-3 建立一個簡單的留言板

# 目標

體驗 Rails 的快速開發。
建立一個留言板：裡面可以建立文章標題、可以填入正文內容。

# 操作

first_app 只是個剛建好的空專案，為了讓大家更能夠感受 Rails 的快速開發，我們現在要加點功能，把它做成一個留言板。

# 1.首先，再開一個新的終端機視窗輸入指令使用

在上一個章節中，你會有一個正在執行的終端機視窗，讓它執行，不要關掉。

我們直接再開一個新的終端機視窗 ( 快速鍵： ⌘ + n )，進行以下操作：

cd first_app 進入到 first_app 的資料夾

ls 確認目前資料夾位置的檔案結構

輸入：

rails g scaffold topic title:string description:text

接著輸入：

rails db:migrate

請確保你此時已經有一個終端機視窗正在執行 server ，沒有的話要新開一個終端機視窗再輸入：

rails s

# 2.體驗留言板功能

體驗剛建好的留言板功能

前往 http://localhost:3000/topics

玩玩看留言板功能吧!

# 開發小知識

Terminal快速鍵：⌘command + N 開啟新視窗。

未來在開發的時候，你應該有兩個視窗：一個放著跑 rails s , 一個繼續做指令操作來開發。

你也可以選擇：⌘command + T ，這個指令可以開啟一個新的標籤頁。
