# Ruby 基礎

# 1. 基本元素: 輸入、計算、輸出

  • 標準輸出
puts 'hello world!'
# hello world
  • 變數
a = 1
a = '123'
  • 字串內變數 "#{x}"
puts "hello #{a}"
# hello 123
  • 資料類型判斷方式 class xxx
1.class
# => Integer
1.0.class
# => Float
'hello'.class
# => String
[1,2].class
# => Array
{a: 1, b: 2}.class
# => Hash
nil.class
# => NilClass
true.class
# => TrueClass
false.class
# => FalseClass
:hello.class
# => Symbol
  • 資料型態轉換
# [數字 -> 字串]
1.to_s # => "1"


# [字串 -> 整數]
'1'.to_i # => 1

# [字串 -> 浮點數]
'1.1'.to_f # => 1.1

# [無條件捨去 / 無條件進位 / 四捨五入 ]
10.555.floor # => 10
10.555.ceil # => 11
10.555.round # => 11
10.555.round(1) # => 10.6
10.555.round(2) # => 10.56

    • 關係運算子 == != < > <= >=
    • 邏輯運算子 && || !
  • Control Flow 控制流程
    • if else
# [if-else]
if 1 > 2
  puts 'A'
elsif 3 >= 5
  puts 'B'
else
  puts 'C'
end

# [case-when]
a = 3
case a
when 1
  puts 'A'
when 3..5, 7
  puts 'B'  
else
  puts 'C'
end

# 8. 高階函式

接下來提到的高階函式，都使用以下宣告的 array 來講解

array = [
  {a: 1, b: 2},
  {a: 1, b: 3}
]

# select vs filter()

找出 Array 中符合條件的 資料集合

  • 有兩種寫法
    • 單行
    • 多行
  • 習慣
    • 通常能寫單行以單行為主，多行通常是因為太長不好看
# [不縮寫]
array.select do |x|
  x[:a]==1
end
# => [{:a=>1, :b=>2}, {:a=>1, :b=>3}]

# [縮寫]
array.select{|x| x[:a]==1}
# => [{:a=>1, :b=>2}, {:a=>1, :b=>3}]

# find vs find()

找出 Array 中符合條件的 首筆資料

# [不縮寫]
array.find do |x|
  x[:a]==1
end
# => {:a=>1, :b=>2}

# [縮寫]
array.find{|x| x[:a]==1}
# => {:a=>1, :b=>2}

# each vs forEach()

單純走訪所有資料，不回傳(return)

# [不縮寫]
array.each do |x|
  puts x[:a]
end
# 1
# 1

# [縮寫]
array.each{|x| puts x[:a]}
# 1
# 1

# map vs map()

走訪所有資料，並將回傳值組成一個新陣列

# [不縮寫]
array.map do |x|
  x[:a]
end
# => [1, 1]

# [縮寫]
array.map{|x| x[:a]}
# => [1, 1]

# all? vs every()

所有走訪的對象都符合條件的話，得到 true，否則 false

# [不縮寫]
array.all? do |x|
  x[:b]>2
end
# => false

# [縮寫]
array.all?{|x| x[:b]>2}
# => false

# any? vs some()

任一個走訪的對象符合條件的話，得到 true，否則 false

# [不縮寫]
array.any? do |x|
  x[:b]>2
end
# => true

# [縮寫]
array.any?{|x| x[:b]>2}
# => true

# reduce vs reduce()

可設定一個初始值，每次走訪會與前一個回傳的值再次作運算

# [不縮寫]
array.reduce(0) do |init, x|
  init + x[:a]
end
# => 2

# [縮寫]
array.reduce(0){|init, x| init+x[:a]}
# => 2
