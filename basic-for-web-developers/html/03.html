# 三、表格

# 1. 基本概念

  • 表格是由 <table>...</table> 所包含的區域
  • 表格中有 <thead> 可以表示『標題區塊』
  • 表格中有 <tbody> 可以表示『內容區塊』
  • <tr> 表示『行』
  • <td> 表示『普通儲存格』
  • <th> 表示『標題儲存格』(字體加粗)

# 2. 基本結構

<table>
  <!--表格的標題區塊-->
  <thead>
  　<tr><!--第一行(標題)-->
  　　<th>標題1</th>
  　　<th>標題2</th>
  　</tr>
  </thead>

  <!--表格的內容區塊-->
  <tbody>
  　<tr><!--第二行(內容)-->
  　　<td>文字1-1</td>
  　　<td>文字1-2</td>
  　</tr>
  　<tr><!--第三行(內容)-->
  　　<td>文字2-1</td>
  　　<td>文字2-2</td>
  　</tr>
  </tbody>
  
</table>
標題1 標題2
文字1-1 文字1-2
文字1-1 文字1-2

TIP

  • table: 表格
    • thead: 表格的標題區塊
      • tr: 行
        • th: 儲存格(標題，粗體效果)
    • tbody: 表格的內容區塊
      • tr: 行
        • td: 儲存格(內容，無效果)

# 3. 合併儲存格

<table>
　<tr>
　　<td>1</td>
　　<td>2</td>
　</tr>
　<tr>
　　<td colspan="2">3</td> <!--往右合併2格-->
　</tr>
　<tr>
　　<td rowspan="2">4</td> <!--往下合併2格-->
　　<td>5</td>
　</tr>
　<tr>
　　<td>6</td>
　</tr>
</table>
