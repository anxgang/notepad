# 四、表單

# 1. 基本概念

  • 表單是動態網站的基礎
  • 表單是由 <form>...</form> 包含的區域
  • 表單中不能存在另一個表單

# 2. 基本結構

<form action="xxx.htm" method="get">
  <input type="text">
  <input type="submit">
</form>

# 3. 表格標籤

# form

<form action="xxx" method="post"></form>
  • action：前往網頁 xxx
  • method：執行方法，如：get、post(預設為get)
  • enctype：編碼類型，如要上傳檔案需要加上：enctype="multipart/form-data"

# input

 <input type="text" name="xxx">

# textarea

<textarea name="comment">
  第一行
  第二行
  第三行
</textarea>
  • cols：行數
  • rows：列數
  • disabled：無作用

# radio input

<label><input type="radio" name="color" value="blue" checked>藍色</label>
<label><input type="radio" name="color" value="green">綠色</label>
  • checked：選擇的項目

# checkbox input

<label><input type="checkbox" name="color" value="blue" checked>藍色</label>
<label><input type="checkbox" name="color" value="green">綠色</label>
  • checked：選擇的項目

# select

<select name="car">
  <option value="volvo">Volvo</option>
  <option value="saab" selected>Saab</option>
  <option value="opel">Opel</option>
  <option value="audi">Audi</option>
</select>
  • disabled：沒有作用
  • multiple：允許多重選取
  • selected：選擇的項目
