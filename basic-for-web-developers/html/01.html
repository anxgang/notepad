# 一、初探

# 1. 基本概念

  • 標記式語言 (markup language)
  • 寫法： <標籤名稱> 標籤內容 </標籤名稱>
  • 不分大小寫
  • 最基本的標籤
    • 最外層的 html
    • 上方的 head
    • 主體的 body
  • 可視範圍
    • 可見： 在標籤<body></body>以內，都可以顯示在網頁上
    • 不可見： 在標籤<body></body>以外，不會顯示在網頁上，只是一些設定值

# 2. 基本架構

以下是一個 html5 網頁最基本的架構

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>網頁的標題</title>
</head>
<body>
  網頁的內容
</body>
</html>
  • <!DOCTYPE html>: 宣告 Html 的版本（此寫法表示使用 html5）
  • <html lang="en">(略)</html>: html 開始，帶有一個 lang 的屬性是 en
  • <head>(略)</head>: head 裡面可以設定一些網頁的設定值
  • <body>(略)</body>: body 裡面是網頁真正的內容

# 3. 標籤結構

<a href="https://www.google.com">Visit Google!</a>
  • <a></a>: 標籤，因為寫『a』又代表著它是一個超連結的ㄎ標籤
  • href="https://www.google.com": 標籤的屬性及屬性內容
  • Visit Google!: 標籤的內容
