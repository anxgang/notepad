# 一、初探

# 1. 基本概念

  • 樣式表語言 (stylesheet language)
  • 幫標記式語言增加樣式的語言

# 2. 基本語法

選擇器 {
  屬性: 設定值;
  ...
}

例如：

先在 css 定義好 .hello 的 顏色 和 文字大小

.hello {
  color: red;
  font-size: 40px;
}

在 html 可以直接使用 hello 這個 class 來應用這些樣式

<div class="hello">
  hello
</div>
hello

# 3. 載入方式

  • 行內套用 (Inline)
  • 嵌入套用 (Embed)
  • 外部連接套用 (External Link)
  • 匯入套用 (Import)

# 行內套用

<p style="font-size:20px;">This is font size 20.</p>

This is font size 20.

# 嵌入套用

<style>
  p {
    font-size:20px;
  }
</style>

<p>This is font size 20.</p>

This is font size 20.

# 外連套用

<header>
  <link rel=stylesheet type="text/css" href="external-stylesheet.css">
</header>

<p>This is font size 20.</p>

This is font size 20.
